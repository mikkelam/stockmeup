## Installing
This project uses pnpm. If you want correct dependencies, pnpm must be used.
Installing: 
```bash
pnpm install
```

## Development server
```bash
pnpm run dev

# or start the server and open the app in a new browser tab
pnpm run dev -- --open
```

## Building 

```bash
pnpm run build
```
